# MJH Battle Royale Frenzy

🔥 A fast‑paced, PUBG‑style 2D battle royale shooter built with HTML5 Canvas and JavaScript.

## 📌 Features

- 2D side‑scrolling battle royale gameplay with a shrinking **blue safety zone**.
- Multiple weapons: **Pistol, UMP45, AKM, Shotgun** – each with unique damage, ammo, and fire rate.
- Loot system:
  - Ammo boxes.
  - Medkits that heal your health.
  - Armor (helmet & vest) that reduces damage.
- Enemies patrol and shoot at you with AI‑driven movement.
- Smooth animations, particles, explosions, and a dark, cinematic background.
- Save your **all‑time high score** using browser local storage.
- Can be played in browser **online** and **offline** after PWA installation.

---

## 🎮 How to Play

### Controls (Mobile & Desktop)

- **Drag** the player left/right to move.
- **Tap** the player to **shoot** or **jump** (one tap to jump, second tap to shoot in the same tap).
- Works on mobile (touch) and desktop (mouse).

### Game Mechanics

- Kill enemies to increase your **kill count**.
- Collect **ammo**, **medkits**, and **armor** to survive longer.
- Stay inside the **blue zone**; moving outside deals continuous damage.
- Survive the **shrink** of the safe zone and try to reach **Match 10 / Phase 10** for a “Chicken Dinner!”

## 🚀 Getting Started

### Prerequisites

- A modern browser (Chrome, Edge, Firefox recommended).
- Internet to load the game (unless you install it as a PWA).

### Running the Game

1. Place all files in a web server folder or local server (like `http-server`, `Live Server`, etc.).
2. Open `index.html` in your browser.
3. The game will start on the **Start Screen**.
